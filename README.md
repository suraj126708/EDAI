# 🚪 DoorstepDonation – Donate with Convenience, Impact with Purpose

DoorstepDonation is a web platform designed to make the process of donating everyday items seamless and rewarding. Built using **HTML, CSS, JavaScript, and Node.js**, this project enables users to donate goods like shoes, bags, and stationery to NGOs or the needy — all from the comfort of their homes.

## 🌟 Features

- 🏠 **Doorstep Pickup** – Volunteers collect items directly from the donor's location.
- 🎁 **Reward System** – Donors receive rewards or coupons as a token of appreciation.
- 🧾 **Donation History** – Track what you've donated and when.
- 🧍‍♂️ **Volunteer System** – Volunteers can sign up to manage donation pickups.
- 📞 **Contact & Support** – Easy communication between donors, NGOs, and volunteers.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/suraj126708/EDAI.git

# Navigate into the project directory
cd EDAI

# Install dependencies
npm install

# Start the server
node app.js
````

Open your browser and go to `http://localhost:3000` to view the website.
